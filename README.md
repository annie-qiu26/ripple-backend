## Project Setup
Clone repository:  
```$ git clone https://github.com/annie-qiu26/ripple-backend.git```  
Install from Pipfile:  
```$ pipenv install```  
Activate the Pipenv shell:  
```$ pipenv shell```  
Run Flask app:  
```$ python example.py```  
Deactivate the Pipenv shell:  
```$ exit```  

## Adding packages
```$ pipenv install <package>```
